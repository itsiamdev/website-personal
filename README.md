## 🛠️ Technologies & Tools Used

This project leverages the latest web standards and best practices:

- **🌐 HTML5**: Semantic markup and robust structure for accessibility and SEO
- **🎨 CSS3**: Responsive styling with dedicated stylesheets for each page, ensuring a polished look across devices
- **⚡ JavaScript**: Interactive features such as a mobile-friendly hamburger menu for enhanced user experience
- **📊 Google Analytics**: Integrated tracking for website analytics and performance insights
- **🔍 SEO Optimization**: Comprehensive meta tags, Open Graph, and Twitter Cards for optimal social media sharing
- **🔒 Security**: Implemented Content Security Policy (CSP) and referrer policy for enhanced security
- **🎯 Performance**: Optimized for fast loading and smooth interactions
- **📱 Responsive Design**: Mobile-first approach ensuring compatibility across all screen sizes
- **🎨 Boxicons**: Beautiful iconography for a modern aesthetic
